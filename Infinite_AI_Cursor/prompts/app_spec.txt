## Project Name
TODO App

## Description
Build a fully functional todo application with a clean, modern interface.
Support task creation, completion, editing, deletion, and filtering.

## Tech Stack
- Frontend: React with Vite, Tailwind CSS
- Backend: Node.js with Express
- Database: SQLite (optional, or localStorage for simplicity)

## Core Features
1. Add new todo items
2. Mark items as complete/incomplete
3. Edit existing items
4. Delete items
5. Filter: All / Active / Completed
6. Persist data (localStorage or DB)
7. Responsive design

## UI Requirements
- Clean, minimal design
- Dark mode support
- Smooth animations
- Accessible (keyboard navigation)

## API Endpoints (if using backend)
- GET /api/todos - List all todos
- POST /api/todos - Create todo
- PUT /api/todos/:id - Update todo
- DELETE /api/todos/:id - Delete todo
